# Daily Provider Rounding Checklist - Kidney Transplant

Use this checklist during daily multidisciplinary rounds for adult deceased donor kidney transplant recipients.

## Daily Review
- [ ] Review vital signs and temperature trends
- [ ] Urine output (hourly trends if Foley in place)
- [ ] Daily weights and fluid balance (input/output)
- [ ] Review labs: CBC, BMP, Tacrolimus/Cyclosporine level, LFTs, Mg/Phos
- [ ] Immunosuppression doses and levels
- [ ] Foley status (remove if appropriate)
- [ ] Pain control (adequate, renal-safe meds)
- [ ] Nutrition and GI function (PO intake, bowel status)
- [ ] Signs of rejection (DSA if ordered, biopsy findings)
- [ ] Wound/incision appearance
- [ ] Infection signs (cultures, antibiotics, WBC trends)
- [ ] Prophylaxis compliance (CMV, PCP, fungal)
- [ ] Patient education progression (meds, hygiene, follow-up)
- [ ] Discharge readiness checklist (labs, appointments, meds)
