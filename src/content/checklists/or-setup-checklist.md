# OR Setup Checklist - Kidney Transplant

## Operating Room Setup
- [ ] Slush machine with ice (if first thing to be set up)
- [ ] Bench table
- [ ] Small back table
- [ ] Bench tray
- [ ] Basin with cold slush (kidney stays in slush until first anastomosis)
- [ ] 2-0 and 3-0 silk ties (short or cut in half)
- [ ] Back table
- [ ] Gen Major, Salzberg/Cronin Extras, and Bookwalter with segmented rings

## Equipment
- [ ] Slush machine
- [ ] Fluid warmer
- [ ] IV pole
- [ ] Electrosurgical unit
- [ ] Dornoch
- [ ] Urine counting Foley
- [ ] Argon bovie (available)
- [ ] Doppler (available)

## Medications for Circulator
- [ ] 1000 mL of saline with bottle of betadine inside (to be hooked to Foley)
- [ ] 5000 units of heparin in 500 mL of saline to back table
- [ ] TAPP

## Hemostatics
- [ ] Snow
- [ ] Vistaseal

## Pre-Operative Phase - Verify Documentation
- [ ] Recipient ABO in EPIC
- [ ] Donor ID
- [ ] Surgical consent
- [ ] Blood products consent
- [ ] Verify the patient has a blood band in pre-op
- [ ] Anesthesia consent

## Intraoperative Phase - Prior to Incision
- [ ] Place a 22 Fr 3-way Foley and hook to urine counter
- [ ] Hooked up to betadine irrigation and clamped off
- [ ] Have catheter plug for end of procedure
- [ ] Anesthesia will need A-line and possibly central line
- [ ] Abdominal prep: shave xiphoid to pubis
- [ ] Prep with Chloraprep
- [ ] Incision will be lower left abdomen (Rutherford-Morrison)

## General Steps to Kidney Transplant (for Circulator)
- [ ] Iliac vein and artery located
- [ ] Vein anastomosis first ("off ice") - uses 5-0 Prolene
- [ ] Artery anastomosis second ("clamps off" / organ reperfusion) - uses 6-0 Prolene
- [ ] ~10 minutes before reperfusion anesthesia will be directed to give mannitol 12.5 g IV
- [ ] Will also ask for 20 mg Lasix IV if the patient does not make urine
- [ ] Ureter anastomosis
- [ ] Will place 5fr x 12 cm JJ ureteral stent
- [ ] Uses 5-0 PDS RB-1 for ureter
- [ ] Obtain hemostasis
- [ ] Have Vistaseal
- [ ] Will place a drain (#10 JP)
- [ ] Closure: #1 PDS TP-1 (non looped) for fascia
- [ ] Skin: staples or 4-0 Monocryl
- [ ] Dressing

## Intraoperative Time-Outs
- [ ] Pre-induction time out (completed on arrival to OR, prior to induction of general anesthesia)
- [ ] Verify donor: donor ID
- [ ] Verify donor: organ/kidney laterality
- [ ] Verify donor: blood type
- [ ] Verify recipient: unique identifier (MRN, DOB)
- [ ] Verify recipient: blood type
- [ ] Verify donor and intended recipient are ABO compatible
- [ ] Verify histocompatibility in chart
- [ ] Time-out two: verify donor ABO and donor ID
- [ ] Time-out two: verify organ type/kidney laterality
- [ ] Time-out two: verify recipient ABO and MRN
- [ ] Time-out two: verify ABO compatibility
- [ ] Must be done prior to anastomosis with recipient and organ inside OR
- [ ] Surgeon and circulator must sign in EPIC

## Intraoperative Documentation (Recorded in EPIC)
- [ ] Patient arrival to OR
- [ ] Organ arrival to OR
- [ ] Donor clamp time
- [ ] Pre-induction time out (Time Out 1)
- [ ] Intraoperative time out two (with patient and organ in room)
- [ ] Organ removed from ice (1st anastomosis)
- [ ] Clamps removed (organ reperfusion)
